Student Management System

A web-based Student Management System built using Java, JDBC, JSP, and Servlets on IntelliJ IDEA. This application allows efficient management of student information and attendance with full CRUD operations. It provides a simple and interactive interface to add, update, view, delete, and track students’ attendance records.

Features

Add, view, edit, and delete student details

Manage and update student attendance

View attendance history for each student

Search and filter student records

JSP-based UI and Servlets for backend logic

MySQL database connectivity through JDBC

Technologies Used
Technology	Description

IntelliJ IDEA	IDE for development

Database Structure
Tables

students — stores student information

attendance — stores daily attendance records

How to Run the Project

Clone or download the project

Open in IntelliJ IDEA

Configure Apache Tomcat server

Create a MySQL database and import SQL tables

Update DB credentials in DBConnection.java

Run project and open browser:

http://localhost:8080/StudentManagement/
